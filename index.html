Artist's Statement

This piece serves as a central hub for all upcoming projects. It is represented through an image of tangled earbuds. Each individual earbud links to an existing or future piece. Future projects have a filler web page that notes the project number and notifies that the piece is "Coming Soon." Otherwise, if the project does exist, the link goes to a web page that displays such piece. Currently, the only "working" link is for project one, or the Maze or Labyrinth? piece. There are a total of six links. The link for project two is omitted because the central hub is project two. Earbuds were chosen as the map because of its randomness and lack of direction. When you put down three pairs of earbuds and leave them overnight, you are going to come back to them tangled. The input plug is the starting point, the cords act as the route, and the earbud itself represents the destination. The connection between a set of earbuds will be established with upcoming projects. In other words, each pair of earbuds will have a theme that connects them. For example, project one and project six will have similarities and project three and project seven will also have similarities. This will reinforce the notion that if you begin at one spot, you can accomplish the same goal with a different result. Functionally, this piece was created using Photoshop, paint.NET, and HTML/CSS. The image of the tangled earbuds was created using paint.NET. Then the image was broken up into links using Photoshop's slice tool. The web page was developed with simple HTML and CSS and utilized the open source css library Hover-Master. Code for the image was taken from the html file processed by Photoshop and placed into the custom index.html file created for the central hub.
